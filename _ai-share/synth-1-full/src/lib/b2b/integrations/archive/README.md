| Платформа | Регион | Причина архивации |
|-----------|--------|-------------------|
| JOOR | US/EU | B2B платформа, санкции/ограничения |
| NuOrder | US | Оптовые заказы, US wholesale |
| Fashion Cloud | EU | api.fashion.cloud, EU |
| SparkLayer | UK | docs.sparklayer.io, UK B2B |
| Colect | EU | Лукбуки, docs.colect.io |
| Zedonk | JOOR ecosystem | api-docs.jooraccess.com |

Для РФ: Wildberries, Ozon, Яндекс Маркет — в основных интеграциях.
