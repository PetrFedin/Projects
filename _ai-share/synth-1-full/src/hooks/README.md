# Общие хуки для кабинета клиента

## useUserOrders

Централизованный хук для загрузки и управления заказами пользователя.

**Преимущества:**

- Единая точка доступа к данным заказов
- Автоматический расчет статистики
- Кэширование данных
- Обработка ошибок

**Использование:**

```tsx
const { orders, stats, loading, error, refreshOrders } = useUserOrders();

// stats содержит:
// - totalSpent: общая сумма потраченных средств
// - totalOrders: количество заказов
// - avgOrderValue: средний чек
// - returnRate: процент возвратов
// - recentOrders: последние заказы
// - ordersByStatus: заказы по статусам
```

## useUserActivity

Хук для расчета статистики активности пользователя.

**Преимущества:**

- Объединяет данные из разных источников
- Автоматический расчет уровня вовлеченности
- Единый источник данных для всех компонентов

**Использование:**

```tsx
const activity = useUserActivity();

// activity содержит:
// - wishlistCount: количество в избранном
// - lookboardsCount: количество лукбордов
// - cartCount: количество в корзине
// - ordersCount: количество заказов
// - totalActivity: общая активность
// - engagementLevel: уровень вовлеченности (0-100)
// - loyaltyPoints: бонусные баллы
// - loyaltyPlan: план лояльности
```

## Общие компоненты
