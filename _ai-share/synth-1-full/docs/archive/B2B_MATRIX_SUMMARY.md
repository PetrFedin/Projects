## 🔗 СИНХРОНИЗАЦИЯ С РОЛЯМИ И ПРОФИЛЯМИ

### **Источник данных: useUserContext()**

```typescript
const userContext = useUserContext();
const {
  isRetailer,     // ← Определяет primary role
  isBrand,
  isBuyer,
  currentOrg,     // ← Организация (название, лого, кредитный лимит)
  user            // ← Пользователь (имя, email, роли)
} = userContext;
```

### **Алгоритм работы**:

1. **Определение Primary Role**:
   ```typescript
   if (isRetailer) → 'retailer' (11 функций)
   if (isBrand) → 'brand' (4 функции)
   if (isBuyer) → 'buyer' (9 функций)
   ```

2. **Получение прав доступа**:
   ```typescript
   const allowedItems = ROLE_PERMISSIONS[primaryRole];
   // Например: ['market-intelligence', 'smart-recommendations', ...]
   ```

3. **Фильтрация функций**:
   ```typescript
   const filteredItems = WORKSPACE_ITEMS
     .filter(item => item.category === activeTab)      // По выбранной категории
     .filter(item => allowedItems.includes(item.id));  // По правам роли
   ```

4. **Отрисовка карточек**:
   - Только функции из `filteredItems`
   - Сортировка по приоритету (critical → high → medium → low)
   - Анимации появления

### **Связи с профилем**:

| Данные | Где используется | Откуда |
|--------|------------------|--------|
| Имя пользователя | Role Context Card | `user.displayName` |
| Название организации | Role Context Card | `currentOrg.name` |
| Роль | Определение primary role | `isRetailer/isBrand/isBuyer` |
| Кредитный лимит | Метрики в "Платежный Хаб" | `currentOrg.creditLine` |
| Loyalty tier | Метрики в "Статус Лояльности" | `currentOrg.loyaltyTier` |

```typescript
  WORKSPACE_ITEMS,        // ← 14 функций

} from "@/lib/data/b2b-workspace-matrix";

// ✅ Один цикл отрисовки:
{filteredItems.map((item) => (
  <Card key={item.id}>
    {/* Универсальная карточка */}
  </Card>
))}

// ✅ Названия категорий из данных:
{WORKSPACE_TABS.map(tab => (
  <button>{tab.label}</button>
))}
```

**Преимущества**:
- ✅ **0 дублирования**: один source of truth
- ✅ **Легко добавить**: просто добавь в WORKSPACE_ITEMS
