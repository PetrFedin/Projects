## Чего нет или мало — нужно добавить

### Faire
- **Net terms / оплата с отсрочкой** — типичные 60 дней для ритейлеров
- **First order discount** — скидка на первый заказ
- **Return insurance** — страховка возвратов
- **Faire Direct** — прямой канал бренд→магазин (без маркетплейса)

### Ankorstore (EU B2B)
- **Мультибрендовый заказ в одной корзине** — заказ у нескольких брендов одним чеком
- **Консолидация отгрузок** — одна посылка от нескольких брендов
- **Payment terms (Net 30/60)** — условия оплаты
- **European VAT** — работа с НДС EU

### FashionGo (LA Fashion District)
- **Trade show calendar** — календарь выставок (события по датам)
- **Multi-vendor marketplace** — маркетплейс с множеством брендов
- **Showroom appointments** — бронирование визитов в шоурум

### Shopify Plus B2B
- **Company accounts** — корпоративные аккаунты (несколько пользователей)
- **Volume / quantity breaks** — скидки по объёму (уже есть volume-discounts)
- **B2B checkout** — отдельный checkout с реквизитами компании
- **Tax exemptions** — освобождение от налога для B2B

### BigCommerce B2B
- **Customer groups & price lists** — группы клиентов и привязка прайсов
- **Approval workflows** — многоуровневое согласование (частично есть)

### NetSuite SuiteCommerce
- **ERP-интеграция** — синхрон с NetSuite (заказы, остатки, клиенты)
- **Multi-location inventory** — остатки по складам/локациям
- **Advanced reporting** — отчёты по заказам, продажам, прогнозам

### OroCommerce
- **B2B customer segments** — сегменты B2B-клиентов
- **Multi-website** — несколько сайтов/брендов в одном Oro
- **RMA (returns)** — возвраты и рекламации с workflow

### Alibaba
- **Trade Assurance** — защита сделки (эскроу)
- **Supplier verification badges** — верификация поставщиков
- **Multi-language** — мультиязычность каталога
- **MOQ/price negotiation** — переговоры по MOQ и цене

### Brandboom (дополнительно)
- **Line sheets with zoom** — лайншиты с увеличением
- **Buyer onboarding questionnaire** — анкета при онбординге байера

### Fashion Cloud (дополнительно)
- **Stock bulk API** — массовая выгрузка остатков (есть route)
- **PIM → каталог** — синдикация контента в каталог байера

## Рекомендуемый приоритет для профилей

### Brand
1. Net terms / Payment terms (Faire, Ankorstore, BigCommerce)
2. Company accounts (Shopify Plus)
3. Customer groups + price lists (BigCommerce)
4. RMA / возвраты с workflow (OroCommerce)

### Shop / Buyer
1. Multi-brand cart (Ankorstore)
2. Net terms (Faire)
3. First order discount (Faire)
4. Trade show calendar (FashionGo)

### Supplier / Manufacturer
1. Trade Assurance / escrow (Alibaba)
2. Supplier verification (Alibaba)
3. Multi-location inventory (NetSuite)
