### До

```
┌─────────────────────────────────────┐
│ Header                              │
├─────────────────────────────────────┤
│ [Раздел 1] [Раздел 2] [Раздел 3]   │
├─────────────────────────────────────┤
```

```
┌─────────────────────────────────────┐
│ Header                              │
├─────────────────────────────────────┤
│ [Группа 1] [Группа 2] [Группа 3]   │ ← Группы разделов
├─────────────────────────────────────┤
│ [Раздел 1] [Раздел 2] [Раздел 3]   │ ← Основные разделы
├─────────────────────────────────────┤
│ [Подраздел 1] [Подраздел 2] ...    │ ← Подразделы (если есть)
├─────────────────────────────────────┤

└─────────────────────────────────────┘
```

## Вопросы и ответы

**Q: Можно ли использовать Tabs внутри страницы?**
A: Да, если это функциональные табы для переключения контента (например, "Стримы/Видео/Сторис"), а не дублирующая навигация.

**Q: Как добавить новую группу разделов?**
A: Добавить новый объект в массив `navGroups` в соответствующем файле навигации.

**Q: Нужно ли создавать layout.tsx для подразделов?**
A: Нет, layout уже есть на уровне кабинета. Дополнительные layouts не нужны.

**Q: Как фильтровать данные по подразделам?**
A: Использовать `useSearchParams()` для чтения параметров из URL.
