# Следующие 10 пунктов для развития MVP

Рекомендуемый порядок приоритетов.

---

## 1. Реальный API для профиля бренда

Подключить backend-эндпоинты вместо моков: `/brand/profile/dashboard` уже вызывается через `fastApiService.getBrandProfileDashboard()`. Добавить реальные роуты на FastAPI, чтобы данные (retailersCount, openB2bOrders, integrations и т.п.) приходили с сервера.

---

## 2. Исправить production/page.tsx

Текущая ошибка парсера: `Unexpected token ProductionRoot`. Связано с `return (` + JSX. Исправить вложенность JSX или заменить на паттерн `(null as React.ReactNode) ?? <ProductionRoot>...`, чтобы SWC корректно парсил JSX.

---

## 3. Обработка ошибок и retry

- Оборачивать остальные страницы (B2B, Retailers и т.д.) в `ErrorBoundary`
- Добавить повторные запросы (retry) в `fastApiService` для критичных эндпоинтов
- Показывать тосты при ошибках действий (сохранение, смена статуса)

---

## 4. React Query (TanStack Query)

Ввести `@tanstack/react-query` для кеширования, фоновой синхронизации и invalidation. Заменить `useBrandProfileData` на `useQuery`, подключить `QueryClientProvider` в layout.

---

## 5. Роут-защита и доступ по ролям

Проверять роль и права в middleware или layout `/brand`, `/admin`, `/factory`. Редирект на 403 или на подходящий раздел, если у пользователя нет доступа.

---

## 6. Интеграции 1С / CDEK / Ozon — реальный статус

- Эндпоинты статуса интеграций
- Логи синхронизации
- Кнопка ручной пересинхронизации
- Обновлять блок интеграций на странице профиля из API

---

## 7. Маркировка (Честный ЗНАК)

- Реальный статус ЭДО и синхронизации остатков
- Алерты при расхождении КИЗ
- Связка с `/brand/compliance` и `/brand/compliance/stock`

---

## 8. Экспорт профиля в PDF/Excel

Реализовать генерацию и скачивание PDF/Excel по профилю (DNA, контакты, сертификаты). Сейчас кнопка «Экспорт» только показывает toast. Можно использовать `jspdf` или `xlsx`.

---

## 9. E2E-тесты (Playwright)

Установить Playwright и добавить тесты для ключевых сценариев:  
- логин и навигация по профилю  
- B2B-заказы  
- переходы между Retailers, Production, Compliance  

---

## 10. Мобильная оптимизация

- Адаптивность основных экранов (профиль, B2B, ритейлеры)
- Гамбургер-меню в brand layout на мобильных
- Улучшить touch targets для важных кнопок
