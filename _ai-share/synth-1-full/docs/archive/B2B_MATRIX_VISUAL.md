## 🔄 DATA FLOW: ОТ ПРОФИЛЯ К UI

```
┌───────────────────────────────────────────────────────────────┐
│ 1. USER LOGIN                                                 │
│    ↓                                                          │

                             ↓
┌───────────────────────────────────────────────────────────────┐

│    ↓                                                          │
```
