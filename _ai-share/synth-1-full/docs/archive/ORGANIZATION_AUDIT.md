## 🚀 ЧТО ДОБАВИТЬ

### 1. COMPLIANCE & СЕРТИФИКАЦИЯ (новая вкладка в Профиле бренда)
**Зачем**: Все luxury/premium платформы (Farfetch, Mytheresa, RealReal) требуют:
- ISO сертификаты
- GOTS / Oeko-Tex (текстиль)
- FSC (упаковка)
- B-Corp статус
- Сертификаты происхождения (для экспорта)

**Что добавить**:
```
Новая вкладка "Сертификаты" в /brand:
- Загрузка сертификатов (PDF/JPG)
- Срок действия с уведомлениями
- Статус верификации платформой
- Публичный badge для каталога
```

### 2. ПАРТНЕРСКАЯ СЕТЬ (новая секция в Обзоре)
**Зачем**: JOOR, NuOrder, Alibaba.com показывают всю цепочку:
- Производители (ссылка на `/brand/factories`)
- Поставщики материалов (ссылка на `/brand/materials`)
- Логистические партнеры
- Ретейлеры (для B2B2C)

**Что добавить**:
```
Новый виджет "Партнерская экосистема" на /brand/organization:
- Карточки с количеством активных партнеров
- Быстрый доступ к разделам
- Статус интеграций
```

### 3. ДОКУМЕНТООБОРОТ (новый подраздел)
**Зачем**: Lamoda, Wildberries требуют:
- Типовые договоры (B2B, B2C, комиссия)
- Акты приема-передачи
- Спецификации на продукцию
- ЭДО (электронный документооборот)

**Что добавить**:
```
Новая страница /brand/documents:
- Шаблоны документов
- Генератор договоров
- Архив подписанных документов
- Интеграция с Диадок/СБИС
```

### 4. КАНАЛЫ ПРОДАЖ (усиление раздела Настройки)
**Зачем**: Уникальная фича для мультиплатформы:
- B2B (опт, JOOR-style)
- B2C (розница, Shopify-style)
- Marketplaces (Ozon, Wildberries, Lamoda)
- Outlet (распродажи)
- Resale (Vestiaire, RealReal)

**Что добавить**:
```
Новая вкладка "Каналы" в /brand/settings:
- Включение/отключение каналов
- Маржинальность по каналам
- Правила ценообразования
- Настройки синхронизации стоков
```

### 5. NOTIFICATIONS HUB (усиление раздела)
**Зачем**: Все платформы имеют центр уведомлений:
- Истекающие сертификаты
- Незаполненные обязательные поля
- Задачи на согласование
- Системные алерты

**Что добавить**:
```
Новая секция на /brand/organization (верх страницы):
- Alerts & Actions Required
- Цветовая индикация (red/yellow/green)
- Быстрые действия прямо из алерта
```
