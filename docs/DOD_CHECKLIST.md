# Definition of Done — изменения Core

Перед merge изменения, затрагивающие **демо-путь** или **core API**:

- [ ] Обновлён `docs/MVP_CONTRACT.md`, если менялся состав core.
- [ ] Есть/обновлён smoke или integration тест для затронутого сценария.
- [ ] В demo-режиме (без ключей) нет необработанных 500 на core-маршрутах.
- [ ] Если затронут агент — запись в `docs/AGENT_REGISTRY.md`.
- [ ] Нет расширения MVP-scope без явного решения.

Phase 2 изменения: не ломают core smoke-тесты.
